# 45-Day Data Engineering Sprint 🚀
A structured journey from SQL basics to building automated data pipelines.

## 🛠 Projects Completed
### 1. Weather ETL Pipeline (Python)
- **Extract**: Fetches real-time weather data for London, NY, and Tokyo via Open-Meteo API.
- **Transform**: Categorizes temperatures into 'Hot', 'Cold', or 'Mild'.
- **Load**: Appends clean data to a local `weather_data.csv` with timestamps.

### 2. SQL Analytics
- **Device Viewership**: Used `CASE` statements to pivot device logs into business reports.
- **User Engagement**: Calculated "Post Hiatus" using date-casting and aggregations.
